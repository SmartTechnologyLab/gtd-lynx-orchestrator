# 🔄 GTD Lynx Orchestrator

Multi-Agent AI Orchestrator for building a **GTD (Getting Things Done)** mobile app with **[Lynx](https://lynxjs.org/)** (ByteDance).

Uses the **Supervisor Pattern** — a Team Lead agent decomposes tasks, delegates to specialist agents, reviews their work, and manages sprint planning. **Hybrid approval** model: the agent auto-approves routine work while escalating key decisions (architecture, sprint plans) to you in the terminal.

## Architecture

```
                    ┌─────────────┐
                    │  🧠 Team Lead │
                    │  (Supervisor) │
                    └──────┬──────┘
                           │
              ┌────────────┼────────────┐────────────┐
              │            │            │            │
        ┌─────┴─────┐ ┌───┴───┐ ┌─────┴─────┐ ┌───┴────┐
        │ 🏗️ Architect│ │⚛️ FE  │ │ 🎨 UX     │ │📋 GTD  │
        └───────────┘ └───────┘ └───────────┘ └────────┘
```

## Agents

| Agent | Role | Auto-approve |
|-------|------|-------------|
| 🧠 Team Lead | Decompose, delegate, review, sprint plan | code, docs |
| 🏗️ Architect | App architecture, data models, offline-first | ❌ human required |
| ⚛️ Frontend Dev | ReactLynx components, CSS, gestures | ❌ team lead reviews |
| 🎨 UX Designer | Screen flows, navigation, interactions | ❌ team lead reviews |
| 📋 GTD Expert | GTD methodology, workflow engine, contexts | ❌ team lead reviews |

## Hybrid Approval Flow

```
Agent delivers → Team Lead reviews → Decision:
  ├── Auto-approve (code, docs) → ✅ Done
  ├── Human required (architecture, sprint-plan) → ⏸ You decide
  └── Team Lead rejects → Escalate to you → You override or confirm
```

## Quick Start

```bash
# Clone
git clone https://github.com/SmartTechnologyLab/gtd-lynx-orchestrator.git
cd gtd-lynx-orchestrator

# Install
npm install

# Set your Anthropic API key
export ANTHROPIC_API_KEY=your_key_here

# Run the orchestrator
npm run orchestrate
```

## Tech Stack

**Orchestrator:**
- TypeScript + Node.js
- Anthropic Claude API (claude-sonnet-4-5)
- chalk, inquirer, ora (terminal UI)

**GTD App (target):**
- [Lynx](https://lynxjs.org/) (ReactLynx)
- TypeScript
- Zustand (state management)
- Rspeedy (Rspack-based bundler)
- Offline-first architecture

## Approval Configuration

Edit `approvalConfig` in `packages/orchestrator/src/index.ts`:

```typescript
const approvalConfig: ApprovalConfig = {
  mode: 'hybrid',
  humanRequired: ['architecture', 'sprint-plan'],  // You approve these
  agentCanApprove: ['code', 'documentation', 'gtd-workflow', 'ux'],  // Team Lead approves
};
```

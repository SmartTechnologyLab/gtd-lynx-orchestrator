# GTD Lynx App

This directory will contain the GTD mobile app built with [Lynx](https://lynxjs.org/) framework.

The app code will be generated and iterated on by the multi-agent orchestrator.

## Setup (when ready)

```bash
# Initialize Lynx project
npx @anthropic-ai/create-lynx-app@latest .

# Start development
npm run dev
```

## Target Screens

- 📥 **Inbox** — Capture and quick-add tasks
- 📅 **Today** — Daily funnel (Clarify → Engage)
- 📊 **Projects** — Kanban view of multi-step projects
- 🔄 **Review** — Daily/weekly review checklists
